How it works

1
Select your sprite
Tap the drop zone or drag a PNG file onto it. SplitForge checks it's exactly 60×40 pixels before doing anything.
2
Hit Split
The tool uses the HTML5 Canvas API to crop your image locally — left 40×40 and right 20×40 — with zero quality loss.
3
Preview & Download
Both pieces appear as live previews. Tap Download on each to save them as separate PNG files to your device.
